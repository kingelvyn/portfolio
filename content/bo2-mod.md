# BO2 Interactive Zombies Mod

---

### Summary
> BO2 Zombies mod that allows interactive involvement from a streamer's chat (Twitch, Youtube, etc.) by offerring rewards based on donation's. Inspired by a private mod that I saw on a streamer's page.

### Features:

- Spin the "reward wheel"
    - Points (Small/Large)
    - Random weapon from weapon pool (map specific)
    - Random perk from perk pool (map specific)
    - Powerups (Max ammo, insta-kill, etc.)
    - Receive all perks (map-specific)
- Reward notification texts
- Invalid reward reroller / fallback reward
- Twitch integration (Streamer.bot specifically)
